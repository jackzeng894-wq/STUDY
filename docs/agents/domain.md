# Domain Docs

Single-context repo.

## Before exploring, read these

- **`CLAUDE.md`** at the repo root (project context)
- **`docs/`** — design docs and requirements
- **`docs/adr/`** — architectural decision records (if they exist)

If any of these files don't exist, proceed silently.
